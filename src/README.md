# Mergington High School Activities

A comprehensive web application that helps students discover and teachers manage extracurricular activities at Mergington High School.

## Overview

The Mergington High School Activities website provides a modern, user-friendly platform where students can explore various extracurricular opportunities and teachers can efficiently manage student registrations.

## Key Features

### For Students
- **Browse Activities**: View all available extracurricular activities with detailed descriptions
- **Smart Search**: Search activities by name, description, or schedule details
- **Advanced Filtering**: Filter activities by:
  - Category (Sports, Arts, Academic, Community, Technology)
  - Day of the week (Monday through Sunday)
  - Time preference (Before School, After School, Weekend)
- **Activity Details**: See comprehensive information including:
  - Activity descriptions and schedules
  - Current enrollment numbers and available spots
  - List of current participants
- **Responsive Design**: Optimized for both desktop and mobile devices

### For Teachers
- **Secure Login**: Teacher authentication system with role-based access
- **Student Registration**: Register students for activities directly from the interface
- **Real-time Updates**: View current enrollment numbers and participant lists
- **Session Management**: Secure login sessions with automatic validation

## How to Use

### Browsing Activities
1. Visit the website to see all available activities
2. Use the search bar to find specific activities
3. Filter by category, day, or time to narrow down options
4. Click on activities to see detailed information including schedules and current enrollment

### For Teachers
1. Click the "Login" button in the top-right corner
2. Enter your teacher credentials
3. Once logged in, you can register students for activities
4. Use the filtering and search tools to quickly find specific activities

## Activity Categories

The website organizes activities into five main categories:
- **Sports**: Physical activities, teams, and fitness programs
- **Arts**: Creative activities including art, drama, and cultural clubs
- **Academic**: Educational clubs and competition teams
- **Community**: Service and community engagement activities  
- **Technology**: Programming, robotics, and technology-focused groups

## Technical Information

- **Frontend**: Modern HTML, CSS, and JavaScript with responsive design
- **Backend**: FastAPI framework with MongoDB database
- **Authentication**: Secure teacher login system with session management
- **Real-time Features**: Dynamic filtering and search capabilities
